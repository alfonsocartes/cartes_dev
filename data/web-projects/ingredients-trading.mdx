# Ingredients Trading 71

[Ingredients Trading 71](http://www.ingredients71.com) is a company website for a medium business in the food flavouring industry.

The website was the first "real" company website that I made after University with HTML, Kickstart CSS and JQuery.

It has been up and running for more than 9 years.

Ingredients Trading 71 is a Venezuelan Flavours company in the food industry that suplies food ingredients to the local markets.

## Technologies

- HTML / CSS / Javascript
- JQuery
- Kickstart CSS
- LAMP Stack
- Linux Server
