# Pokédex SSG - SSR

Pokédex is a DEMO web app to showcase web development technologies such as:

- ReactJS
- NextJS
- Static Site Generation (SSG)
- Server Side Rendering (SSR)
- RESTful API
- TailWind CSS v2 (with Dark Mode!)
- HTML 5

**Some cool features:**

- [Tailwind Dark Mode](https://tailwindcss.com/docs/dark-mode)
- [Next.js 10 Image Component Optimization](https://nextjs.org/docs/basic-features/image-optimization)
- [Next.js 10 Dynamic Routes](https://nextjs.org/docs/routing/dynamic-routes)

This Web App was made by **Alfonso Cartes**. For more information [click here](https://www.cartes.dev/about)

The source code for the app can be found on [GitHub](https://github.com/alfonsocartes/pokedex-example)

## Live Project

Open [https://pokedex-example-sigma.vercel.app/](https://pokedex-example-sigma.vercel.app/) with your browser to see the result.

## Getting Started

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.

## Learn More

To test Static Site Generation and Server Side Rendering, please take a look at the section in `pages/index.js`.

```bash
# Change here for testing SSR or SSG:
<Link href={
    ssgHref
    # or
    ssrHref
}>
```
