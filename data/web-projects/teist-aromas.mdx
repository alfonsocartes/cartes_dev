# Teist Aromas

[Teist Aromas](http://www.teist.es) is a company website with an e-commerce platform.

Teist Aromas is a Spanish Flavours company in the food industry that supplies food ingredients to Europe and America.

_The e-commerce functionality is currently disabled for business reasons._

The website was developed with Wordpress using the LAMP Stack.

## Technologies

- WordPress
- Woocommerce (disabled at the moment)
- Cloudflare CDN
- LAMP Stack
- Linux Server
- Yoast SEO
- Social Network Connections

The website is made for a Spanish company in the Food Industry. It has a focus on SEO, and for some time, it also provided an e-commerce store using the Woocommerce plugin.
